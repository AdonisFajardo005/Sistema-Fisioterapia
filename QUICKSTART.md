## Ejecutar el sistema en 3 pasos:

### 1. Instalar dependencias (solo la primera vez)
```bash
npm install
```

### 2. Iniciar el servidor
```bash
npm start
```

### 3. Abrir en el navegador
Ve a: **http://localhost:3000**
